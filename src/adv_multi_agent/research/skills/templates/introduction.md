### Introduction structure

Write in four moves (no subheadings — flowing prose):

**Move 1 — Establish territory (1–2 paragraphs).**
Motivate the research area. Start with the domain challenge, not with a citation. Establish that the problem is important and active. End with a sentence that sets up the gap.

**Move 2 — Establish the gap (1 paragraph).**
State what existing work does NOT do. Be specific: name what is missing (a capability, a setting, a scale, a theoretical understanding). Cite 3–5 key prior papers by name to demonstrate awareness of the field.

**Move 3 — Introduce the work (1–2 paragraphs).**
Describe the proposed approach at a high level. No equations, no pseudocode — conceptual clarity only. End with the main result (one number or qualitative outcome).

**Move 4 — List contributions (bullet list).**
State 3–5 crisp contributions starting with verbs: "We propose...", "We demonstrate...", "We release...". Each contribution should be independently verifiable.

**Paper map (1 sentence).**
End with: "The remainder of the paper is structured as follows: Section 2 ..., Section 3 ..., ..."
